System

高度なパーソナライズを実現するため、3つの主要なAI技術を組み合わせています。各カードをクリックして詳細をご覧ください。

🧠 プロンプトエンジニアリング

ユーザーの曖昧な要望を的確な提案に繋げます。

ユーザーの好み、目的、避けたいスタイルなどを自然な対話から引き出すための質問（プロンプト）を最適化します。これにより、AIの応答精度を飛躍的に向上させ、本当に求めているスタイルを発見する手助けをします。

📚 大規模言語モデル (LLM)

ファッションの知識とトレンドを学習します。

膨大なファッション雑誌、ブログ、SNSのデータを学習させることで、専門用語や最新トレンドを理解した自然な対話を実現します。ファインチューニングにより、特定のブランドやスタイルに特化した知識を持つことも可能です。

🖼️ Vision-Language Model (VLM)

画像からスタイルや雰囲気を読み取ります。

テキストと画像を同時に処理できるAIモデルです。ユーザーがアップロードした写真やインスタグラムの投稿から、色、形、素材、スタイルといった視覚的特徴を抽出し、言語データと結びつけて提案に活かします。
